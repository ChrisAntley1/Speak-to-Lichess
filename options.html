Checking token status with Lichess...

A personal Lichess API token is required to use Automatic Submission. Click 'Generate Token' to navigate to Lichess's token creation page. The name and required scope have already been set; simply press the blue 'submit' button and copy the generated token into the field below.*

**Only available in rapid, classical, and correspondence formats!**

WARNING: Do not add additional permissions to your token! Token is stored in plain text!





*If you are not comfortable storing an API token, or if you would like to dictate moves in blitz or faster game speeds, you can opt to use the text input box provided by Lichess instead. The text input box does require you to press 'enter' to submit dictated moves.
